# check-group

> A GitHub App built with [Probot](https://github.com/probot/probot) that A app to group checks based on subprojects.

https://github.com/tianhaoz95/check-group-test

## Setup

```sh
# Install dependencies
npm install

# Compile
npm run build

# Run
npm run start
```
